<h3 align="center">All In One Application</h3>
<h5 align="center">Requirement</h5>
<p align="center">
I was required to develop a video downloader application that would be unique in its design and functionality (Extra functionality compared to existing downloaders).
</p>
<h5 align="center">My Approach</h5>
<p align="center">
Along with downloading functionality I provided videos bucket, This essentially keeps a history of any played video so that users could watch these videos online without going to website or download later. Users can take screenshot of long pages that don't fit into one screen. And few other functionalities. 
</p>
<h5 align="center">My Approach(Technical)</h5>
<p align="center">
I used webview to display websites, Injected javascript code that can scrap download-url and video-thumbnail only if the video is in visible screen. Then this info, along with some other meta-data is stored using room-database. A video player can play this video using download-url or downloader-api can download video.
</p>
